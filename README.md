# Dev

1. Clonar el proyecto
2. Copiar el ```.env.template``` y renombrar a ```.env```
3. Ejecutar
```
yarn start
```
4. Levantar la imagen
```
docker-compose up -d
```
5. Levantar el backend de Nest
```
yarn start:dev
```
5. Visitar el sitio
```
localhost:3000/graphql
```
